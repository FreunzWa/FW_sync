###### BZD abuse

# BZD intoxication
- presentation
    + slurred speech
    + nystagmus
    + incoordination
    + gait ataxia
    + UI
    + resp depression
    + amnesia
    + stupor/ coma
    + paradoxical features (agitation, emotional lability, confusion, tachycardia, HTN, tachypnoea)

# BZD withdrawal
- presentation
    + anxiety
    + insomnia
    + aches/ cramps
    + tremor
    + perceptual distortion/ hallucinations
    + seizures
    + palpitations/ tachycardia
    + depression
    + hyperacusis
